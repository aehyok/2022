- mysql 通过sql替换字符串中某一段
  ```
    // 只替换一种/xxxx.jpg
    UPDATE File 
    SET Url = REGEXP_REPLACE(Url, '/[^/]+\\.jpg$', `Name`)
    WHERE Id=1644689089305382912

    // 可以替换多种类型 /xxxx.jpg /xxxx.png等
    UPDATE File 
    SET Url = REGEXP_REPLACE(Url, '/[^/]+\\.(jpg|png|mp4)$', CONCAT('/', `Name`))
    WHERE Id = 1644689089305382912;
  ```
